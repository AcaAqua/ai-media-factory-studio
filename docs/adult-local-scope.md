# Adult Local Scope

AI Media Factory Studio supports local content scopes such as `general`, `sensitive`, and `adult_local` for organization and storage separation.

## Purpose

Adult Local scope is intended to reduce accidental mixing of local assets by separating storage, library tabs, and output routing.

## What It Does

- Keeps Adult Local assets logically separate from ordinary library views.
- Uses dedicated storage settings.
- Requires Adult Local output storage to be safely under the configured ComfyUI output root before real submission.
- Blocks ComfyUI submission when Adult Local storage is missing, disabled, unavailable, not writable, or outside the output root.

## What It Does Not Do

- It does not perform automatic content classification.
- It does not provide legal classification.
- It does not provide legal advice.
- It does not perform age verification.
- It does not determine regulatory compliance.
- It does not decide whether generated or imported content can be published or shared.
