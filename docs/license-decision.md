# License Decision

## Decision

AI Media Factory Studio adopts the Apache License, Version 2.0.

## Rationale

Apache-2.0 is a permissive license with explicit patent license language. It is a good fit for a local-first developer and creative operations tool that may be integrated into private workflows while keeping third-party models, workflows, and generated assets outside this repository.

## Earlier Candidates

The following candidates were considered before the Apache-2.0 decision:

- MIT
- Apache-2.0
- GPL-3.0
- Commercial dual license

This file records the decision history only. The current license is the `LICENSE` file at the repository root.
